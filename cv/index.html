  • 2024.09 - 2024.12
    Deep Learning-based Dynamical System Solver
    Rethinking deep learning approach for modeling dynamical systems governed by Partial Differential Equations (PDEs) by incorporating both spatial and temporal feature learning.
    • Successfully modeled dynamical systems by adding Mamba state space model to learn the temporal features on top of spatial feature extraction, proposing an alternative approach to traditional numerical PDE solvers.
    • Developed two varients of our model to learn spatial features, one uses CNN and the other uses image patching, inspired by the patching method in vision transformer.

    • Tested our models using the advection equation and Burgers' equation which both model fluid dynamics, with the latter being a nonlinear second-order PDE.
